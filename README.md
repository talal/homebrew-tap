# homebrew-tap

## Installation

```
brew tap talal/homebrew-tap
```

## Usage

```
brew install <formula-name>
```

## Formulae

| Name | Description |
| --- | --- |
| `font-iosevka-ss08` | [Iosevka](iosevka) is an open-source typeface family designed for writing code, using in terminals, and preparing technical documents. |
