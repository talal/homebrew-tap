# homebrew-tap

## How do I install these packages?

`brew install talal/tap/<formula>`

Or `brew install --cask talal/tap/<cask>`

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "talal/tap"
brew "<formula>"
cask "<cask>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
